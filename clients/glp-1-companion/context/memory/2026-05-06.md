- `prisma/schema.prisma` — Added PushSubscription model

### Decisions
- Web Push (VAPID) over Firebase — no external dependency, server-side logic reusable when migrating to React Native
- HMAC-signed action tokens for SW auth — service worker fetch doesn't carry Clerk cookies
- Multi-dose support: count today's logs vs schedule.times.length instead of boolean taken check
- Time parser handles both "17:53" and "1753" formats to match existing data
- Cron set to */5 minutes via system crontab
