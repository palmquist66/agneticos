### Decisions
- Libraries: @kingstinct/react-native-healthkit v14 (iOS, Nitro/New-Arch) + react-native-health-connect v3.5 (Android); no cross-platform wrapper → thin lib/health.ts abstraction
- Dedup via existing `(userId, loggedAt, source)` composite unique (matches Dexcom) rather than a new externalId column; sample UUID kept in metadata for future delete-sync
- Activity (steps/energy) = daily aggregate, re-read trailing 7-day window each sync (idempotent upsert) to avoid partial-day overwrite; weight/glucose = incremental anchored
- Native modules lazy-`require`'d per platform → lib/health.ts typechecks even before `npx expo install`
