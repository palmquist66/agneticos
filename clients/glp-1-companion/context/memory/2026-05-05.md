### Decisions
- Used hex values instead of OKLch for CSS variables (simpler, matches design system tokens.css format)
- Coral (#FF8A6A) as accent color for FAB and GLP-1-specific elements to differentiate from general teal primary
- Safari caching required private window testing — no service worker, just aggressive browser cache
