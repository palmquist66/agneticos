Hey Karen,

A few quick tips for Cowork based on what we talked about.

**1. You probably only need one project for HPAH**

A project is just a workspace that remembers your context. Since everything you're doing is for the practice (GBP posts, processing reports, writing content), it all belongs in the same project connected to your `HPAH-Brand` folder. That's the one we set up with your voice profile, ideal customer, and positioning files.

Only create a new project if you're doing something totally unrelated to the practice, like personal stuff. If it's HPAH work, keep it in the same project so Claude remembers your preferences and brand voice.

**2. Start new conversations often (this is the big one)**

When a conversation gets really long, Claude can start to forget things from earlier in that chat. Easy fix: just start a new conversation inside the same project. Your project instructions, brand files, and saved memories all carry over automatically. You don't lose anything.

Think of it like flipping to a fresh page in a notebook. The notebook (your project) stays the same, but the page is clean. No downside to doing this. I'd say start a new conversation for each new task rather than continuing one long thread.

**3. When you export a file from your practice management software**

Save the exported file directly into your `HPAH-Brand` folder (or a subfolder inside it). That way Claude can see it and process it. Don't try to drag files from Downloads into the chat. Cowork can only read files that are already inside the connected folder.

Those three things will cover most of what you need day to day. Let me know if you have questions!
