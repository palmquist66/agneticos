### Decisions
- Root cause was Python null-safety: MLB Stats API returns `currentTeam: null` (JSON null), and `dict.get("currentTeam", {})` returns `None` not `{}` when key exists with null value. Chained `.get()` on `None` = `AttributeError`. Fixed 6 instances with `(dict.get(key) or {})` pattern.
- Second bug: `stats: []` (empty array) for NPB players. `sdata.get("stats", [{}])[0]` throws IndexError because default only applies when key is missing, not when value is empty. Fixed with explicit empty-check.
