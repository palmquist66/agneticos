# BetBro Beta Growth — Trending Research to Content Pipeline

## Goal
Build BetBro's audience across X, Instagram, YouTube, and Reddit during NBA playoffs + MLB season so we have real followers, viewers, and users by NFL/CFB kickoff (~September 2026).

## What's Running
- [x] Trending research → daily content pipeline (X posts + IG posts + 1 Reel/day)
- [x] X engagement reply farming — 3x daily cron, additive tone, no links
- [x] Post reminders cron
- [x] Feedback sync cron
- [x] Daily data sync (NBA + NHL → Render deploy at 6 AM CT)

## Growth Initiatives

### Phase 1 — Now (April)
- [ ] **YouTube Shorts** — repost daily Reels to YT Shorts for additional reach. Same content, new surface.

- [ ] **Content repurposing pipeline** — one research session → X + IG + Reel script + YT Short + Reddit comment. More surface area, same effort.

### Phase 3 — June/July (pre-football)
- [ ] **Email/SMS list** — owned audience. Every piece of content drives to list, not just the app.
- [ ] **Collaborations** — guest spots on betting podcasts, cross-promos with betting content creators.

## Constraints
- **TikTok is off-limits** — BetBro account was banned immediately for gambling content. Don't invest time here.
- No links in X replies (algorithm suppresses)
- X replies must be additive tone only (never dunk on OP)
- Always cross-reference BetBro app data before endorsing props
- All content must go through humanizer before publishing

## Acceptance Criteria
- Research surfaces real discussions with engagement metrics, not generic topics
- Content matches BetBro's voice (direct, playful, provocative)
- Every piece has a path back to the beta signup (getbetbro.com)
- Measurable follower/viewer growth week-over-week through NFL kickoff
